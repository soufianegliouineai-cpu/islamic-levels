# 🌟 المستويات الإيمانية - PWA

 Progressive Web App for tracking your daily Islamic spiritual progress.

## Features

- 📊 **5 Islamic Levels** with detailed daily tasks
- 💎 **Gems System** - earn gems by completing tasks
- 🔥 **Streaks** - track consecutive days
- 🏆 **10 Achievements** - unlock as you progress
- 🎯 **Daily Challenges** - bonus tasks for extra gems
- 👨‍👩‍👧 **Family Support** - track progress together
- 🌙 **Dark Mode** - easy on the eyes at night
- 📱 **Installable** - add to home screen on any device
- 📴 **Works Offline** - data saved locally
- 🔄 **Auto Sync** - your data persists across sessions

## Install

### On iPhone/iPad (iOS)
1. Open Safari and visit the app URL
2. Tap the Share button (📤)
3. Tap "Add to Home Screen"
4. Tap "Add"

### On Android
1. Open Chrome and visit the app URL
2. Tap the menu (⋮)
3. Tap "Add to Home Screen"
4. Tap "Add"

### On Desktop
1. Click the install icon in the address bar
2. Click "Install"

## Tech Stack

- Pure HTML/CSS/JavaScript (no frameworks)
- Service Worker for offline support
- LocalStorage for data persistence
- Responsive design (works on all devices)

## Development

```bash
# Run locally
npx serve .

# Or use Python
python3 -m http.server 8000
```

## Deployment

Deploy to Vercel:

```bash
vercel --prod
```
